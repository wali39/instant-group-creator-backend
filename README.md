# Instant Group Creator API

## Introduction

We developed RESTFUL API using node.js, express.js and sqlite3

## Tools

- Node.js
- Express.js
- typeorm
- sqlite3
- cors
- debug

### Installation

2. Clone the repo

   ```js
   git clone https://github.com/wali39/instant-group-creator-backend.git
   ```

3. Install NPM packages
   ```sh
   yarn
   ```
4. Run api
   ```sh
   node index.js
   ```
